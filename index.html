logo and title Spelling!

Instructions

  1. An image and definition will appear on the screen with the number of letter of the word
  2. Fill in the letters one by one and they will go green if correct and red if you need to try again
  3. If red delete the letter and try again
  4. The hint will tell you the odd thing about the spelling if you click on it
  5. Do as many words as possible before the timer runs out
  6. If you do all the words .......
  7. You can skip if you can't work it out and if there is time you will be taken back to it
